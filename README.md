
# Pixel Arena Shooter

A fast-paced 2D pixel-art arena game built in Unity, focused on responsive movement, combat mechanics, and (soon) LAN multiplayer gameplay.

---

## Samples

![Gameplay Screenshot](./screenshots/gameplay1.png)

---

## Tech Stack

* **Engine:** Unity
* **Language:** C#
* **Networking:** Netcode for GameObjects (NGO)
* **Version Control:** Git + GitHub

---

## 🧠 Technical Focus

This project explores:

* Real-time player control and responsiveness
* Game loop and state management
* Scalable structure for multiple game modes
* Foundations of multiplayer synchronisation

## 🛠️ Setup

1. Clone this repository:

   ```bash
   git clone https://github.com/FaheemFHM/pixel-arena-game.git
   ```

2. Open the project in Unity

3. Load the main scene and press Play
